Функции

Базовый набор модулей для офисной системы управления работой.

Календарь и расписание

События, повторения, приглашения, напоминания, связь с задачами и проектами.

To-Do и задачи

Статусы, приоритеты, чек-листы, теги, исполнители, наблюдатели, комментарии, файлы.

Канбан

Колонки, WIP-лимиты, правила переходов, шаблоны досок, быстрые фильтры.

Гант и зависимости

Связи задач (FS/SS), критический путь (опционально), план/факт, базовый прогресс.

Отчёты и аналитика

Lead/Cycle time, throughput, burndown, загрузка по людям/проектам, SLA.

Интеграции

Telegram-бот, webhooks, экспорт/импорт/синк в Sheets, уведомления и правила автоматизации.

Дашборды

Отдельные экраны по модулям + общий мониторинг исполнения работ и рисков.

Общий мониторинг

Просрочки, блокеры, загрузка, WIP, тренды, топ-риски, состояние проектов.

Проектный

Канбан + Гант + календарь проекта, зависимости, релизы, KPI по проекту.

Личный

Мои задачи, моя неделя, уведомления, быстрые действия, фокус-режим.

Технологии (вариант)

Если будете разворачивать как продукт: Django/DRF + PostgreSQL + Celery + SPA.

Backend

Python/Django, DRF, PostgreSQL, Celery+Redis, WebSocket/SSE для обновлений.

Frontend

React/Vue + роутинг, компоненты дашбордов, drag&drop канбан, календарные виды.

Desktop

Tauri/Electron (online-first), синхронизация с API, кэш и очередь действий.

Примечание: Google Sheets — зарубежный сервис, возможны ограничения. Для РФ часто практикуют VPN и/или альтернативы (self-host таблицы/BI).
