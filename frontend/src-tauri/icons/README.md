# Иконки для Tauri (Windows)

Для сборки десктопа на Windows нужен файл **icon.ico** в этой папке.

## Быстрый способ

1. Возьмите любое квадратное изображение (PNG или SVG), например логотип 512×512 или 1024×1024.
2. Из корня `frontend` выполните:
   ```bash
   npx tauri icon путь/к/изображению.png
   ```
   Например: `npx tauri icon ../public/logo.png`
3. Команда сгенерирует `icon.ico`, `icon.icns` и другие размеры в этой папке.

## Временный вариант

Скопируйте любой `.ico` файл сюда и назовите его `icon.ico`, либо создайте минимальный через онлайн-генератор (например, convertio.co, favicon.io).
